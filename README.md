# coeficiente-UTFPR

## Simule o coeficiente com futuras notas

Arquivo exemplo: `` document.xls ``

Extensões suportadas: xls, xlsx e ods. 

Graças a: [Rows](https://github.com/turicas/rows)

---

Para rodar:
```
pip3 install -r requirements.txt

python3 ./src/main.py
```
